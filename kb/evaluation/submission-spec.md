# Submission Spec Notes

For challenge packaging, repository artifacts should include:

- `results/` with benchmark JSON outputs.
- `eval/` scripts used to generate and score outputs.
- `probes/probes.md` with adversarial cases and fix evidence.
- `kb/corrections/corrections_log.md` showing learning loop entries.

Team-owned external assets (PR links, leaderboard screenshots, public posts) should be recorded under `signal/` and `results/score_log.md` when available.
