# Signal Directory Guide

This directory contains communication and engagement artifacts for submission support.

## Contents
- `engagement_log.md`: Ongoing engagement records
- `community_participation_log.md`: Community participation evidence
- `external_engagement_summary.md`: External outreach summary
- `demo_checklist.md`: Demo readiness checklist
- `submission_checklist.md`: Final submission checklist

## Purpose
- Provide auditable communication and readiness evidence for facilitators and reviewers.
